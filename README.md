# SatisfactorIIE

A management game where the goal si to collect enough resources with machines and conveyor belts to win. \
One or two players.

## Usage

| Command      | Description      |
|--------------|------------------|
| `make all`   | Build project    |
| `make run`   | Run project      |
| `make clean` | Clear built file |

## Example

```bash
make
make run
```
